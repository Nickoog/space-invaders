# Interludes — Photos personnelles

Dépose ici les photos qui apparaissent entre les niveaux du jeu.

| Fichier         | Après le niveau |
|-----------------|-----------------|
| `interlude1.jpg` | 4              |
| `interlude2.jpg` | 8              |
| `interlude3.jpg` | 12             |
| `interlude4.jpg` | 16             |

Les photos sont affichées centrées sur l'écran avec le message personnalisé en dessous.
Formats acceptés : JPG, PNG, WebP.
Taille recommandée : environ 600 × 400 px (le jeu adapte automatiquement).

Pour modifier les messages → ouvrir `src/interludes.ts`.
